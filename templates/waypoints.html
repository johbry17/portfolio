Waypoints

A Personal BioGeography

Waypoints is a personal exploration of my life's journey, brought to life through interactive Leaflet.js maps, photo reels, and data-driven storytelling. This project combines geographic data with personal narratives to create an engaging and immersive experience. Each waypoint highlights a meaningful place or moment, weaving together my global adventures with the power of visualization.

Project Overview

Waypoints is a deeply personal project that maps my life's journey through geography and storytelling. From the serene beaches of the Caribbean to the vast deserts of Namibia and the vibrant streets of India, each waypoint represents a pivotal experience. This project reflects my passion for exploration and my ability to transform data into compelling narratives, offering a unique perspective on how personal stories can be visualized through technology.

Technically, Waypoints leverages a suite of geospatial tools to create an interactive and visually rich experience:

  • Leaflet.js - For rendering dynamic, interactive maps with features like custom markers, clustering, and polylines for hiking routes.
  • Overpass API - To query OpenStreetMap data for geographic features such as hiking trails, ferry routes, and waterways.
  • Nominatim - For geocoding and reverse geocoding, converting locations into geographic coordinates and vice versa.
  • OpenRouteService API - For routing and geospatial analysis, including isochrones and multi-modal transportation routes.
  • GeographicLib - For precise geodesic calculations, such as great circle routes for airplane travel.
  • geojson.io - A web-based tool for drawing and editing custom geographic data, such as boat and ferry routes, and exporting them in GeoJSON format.
  • GeoJSON - As the primary format for encoding geographic data structures, enabling seamless integration with mapping libraries.

The backend, powered by Python, processes data from multiple sources, including APIs, Google Sheets, and CSV files. Google Sheets serves as a flexible data management layer, inspired by SQL principles. The maps include custom markers, activity layers, and photo popups, creating an intuitive and immersive user experience.

This project showcases my expertise in data engineering, visualization, and full-stack development. By combining advanced techniques like geospatial routing, custom legends, and multilayer toggling, Waypoints demonstrates my ability to create responsive, user-friendly interfaces. It serves as both a personal archive of my travels and a professional example of how technical skills can be used to craft meaningful, interactive experiences.
