DC AirBnB Analysis

Unlocking the Trends of DC's AirBnB Scene

A full-stack development project analyzing AirBnB listings in Washington, DC, integrating PostgreSQL, Flask, Python, JavaScript, and Tableau to create interactive platforms to provide insights into the city's short-term rental market.

Project Overview

This full-stack project aims to analyze the Airbnb market in Washington, D.C., combining backend and frontend technologies to deliver a robust data analysis platform. The project stores listing data in a PostgreSQL database and uses Flask to create a flexible and scalable API. On the front-end, JavaScript builds interactive components, and Tableau provides advanced data visualization, allowing users to explore various trends in the D.C. Airbnb market.

Key metrics such as average nightly rates, listing density by neighborhood, and occupancy rates are made available through a user-friendly interface. By combining Python for backend data processing and Tableau dashboards for data representation, this project effectively communicates complex data patterns in an intuitive manner. Additionally, the backend API allows for smooth data retrieval and updates from the PostgreSQL database. Users can view and filter results based on their interests, such as comparing different neighborhoods or analyzing changes over time.

This project demonstrates my ability to manage the full data lifecycle, from database management to API creation and front-end development, while ensuring that the data is visually accessible and insightful for users, demonstrating a comprehensive approach to understanding Airbnb data and market trends in Washington, D.C.
