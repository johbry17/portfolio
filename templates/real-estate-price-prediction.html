Real Estate Price Prediction

A Machine Learning Classic: The Ames, Iowa Data Set

This project focused on predicting housing prices using the Ames, Iowa dataset, implementing a neural network and a random forest model. The project emphasized feature engineering, hyperparameter tuning, and a comparison of model performance.

Project Overview

This project explores the predictive power of two machine learning models — neural networks and random forest regression — applied to the Ames, Iowa housing dataset. Using residential property sales data, I aimed to predict house prices and evaluate the effectiveness of each model through a detailed performance comparison.

Starting with exploratory data analysis, I prepared the dataset for model training by examining missing values, correlations, and key variables affecting house prices. The neural network model, built using TensorFlow and Keras, underwent hyperparameter tuning to optimize its performance. Simultaneously, I developed a random forest model to provide a traditional regression approach for comparison.

The evaluation included visualizing residuals, scatter plots, and training loss for both models. These comparisons highlighted the unique strengths and challenges each model faced when predicting house prices. The project concluded with an analysis of which model performed better under various conditions, offering a nuanced understanding of how to apply machine learning to real estate pricing.

Translation: This was baby's first machine learning project, and a neural network was overkill for the data set.
