Project Overview

Utilizing the Ames, Iowa housing dataset, I developed and compared a neural network and a random forest model to predict property prices. The project emphasized feature engineering and hyperparameter tuning for the neural network, achieving a strong predictive performance. The random forest model, with its inherent simplicity, provided a valuable baseline. Tools like SQLite for data processing and Keras for model optimization were integral to the project, which ultimately showcased the different strengths and applications of these two machine learning approaches.

Translation: This was baby's first machine learning project, and a neural network was overkill for the data set.
